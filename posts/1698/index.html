方便地在博客中发说说

 

前言

什么是IA(Issues Anywhere) Talk ？

  • IA talk 利用手机微信发布碎片化思想，并同步至博客，基于GitHub Issues，无需任何后端服务。

特性

无论何时何地都能通过手机微信将消息发送至博客，支持文字、图片、视频，文本撰写可采用Markdown格式。当前为初步版本，后续功能请见开发计划

原理

本项目实现了公众号服务器端的代码。详细内容参见我博客和 Hexo Stellar 主题下的讨论。

支持

目前仅支持在 Hexo Stellar 主题使用，通过timeline插件将说说加入文档。

快速开始

安装GitHub App

新建一个GitHub仓库，安装应用wechat-to-issues，只授权你刚创建的仓库。

关注公众号

GitHub App 安装成功后，根据提示关注。

绑定仓库

然后再向公众号发送 di3a仓库名，前缀di3a是指令名。举个例子，我的仓库是blog，我就发送di3ablog，首尾不能有空格。
